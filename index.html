Follow the pattern and test your memory

How to play Simon Game!

  1. Press the START button. Simon will get the first signal. Repeat the signal by pressing the same color tab.

  2. Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color lenses, in order.

  3. Simon will duplicate these first two signals and add one.

  4. If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
